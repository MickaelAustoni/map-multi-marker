Easily create multiple markers on a google map !

== Description ==
Easily create a google map on your wordpress with a simple shortcode and add multiples markers. Many options are available to fully customize your marker and map.

== Installation ==
1. Upload the entire `map-multi-marker` folder(unzip) to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `/Plugins`/ menu in WordPress.
3. Add the shortcode [map-multi-marker] where you want display the map
